The problem set units you find here are duplicates of the problem sets you hand in. Here's how it works:

After a problem set deadline has passed, the Sandbox will update to contain that particular problem set. You can continue entering your code so that you may check it with our grader.

  • You can submit as many times as you want.
  • You can still see whether your code is Correct or Incorrect.
  • You can still see which test cases your code passed or failed when compared with the grader's code.
  • The code you enter here does not count towards your grade. It is merely here as an extra tool for you to play around with.

Have fun in the sandbox!
